解决在Github上的用户验证问题

  1. 进入Github中生成生成token令牌，Setting-> Developer settings-> Personal access tokens-> Tokens(classic)-> Generate new token-> Generate new token(classic)
  2. 保存该token令牌，不保存就找不到了!!
